Апрель 2026
Описание

Random Task Generator это приложение с графическим интерфейсом для генерации случайных задач. Приложение помогает разнообразить повседневные дела тренировки и рабочие задания.

Возможности

Генерация случайной задачи
История всех задач с датой и временем
Фильтрация по типам: учёба спорт работа
Добавление пользовательских задач
Проверка ввода: название задачи не может быть пустым
Сохранение истории в JSON файл
Автоматическая загрузка истории при запуске
Модульные тесты

Установка и запуск

1. Убедитесь что установлен Python 3.7 или выше
2. Скачайте все файлы проекта
3. Запустите приложение командой:
   python main.py
4. Для запуска тестов выполните:
   python -m unittest Tests.test_task_generator

Примеры использования

Пример 1: Генерация задачи

Нажмите кнопку Сгенерировать задачу
На экране появится случайная задача например Прочитать статью [учёба]
Задача автоматически добавится в историю

Пример 2: Добавление своей задачи

В поле Название введите например Выучить Python
Выберите тип задачи: учёба спорт или работа
Нажмите кнопку Добавить
Новая задача станет доступна для генерации

Пример 3: Фильтрация задач

Выберите в фильтре значение спорт
В списке истории останутся только спортивные задачи
Генерация случайной задачи продолжает работать со всеми задачами

Пример 4: Очистка истории

Нажмите кнопку Очистить историю
Подтвердите действие в диалоговом окне
Вся история задач будет удалена


Требования

Python 3.7 или выше
Tkinter входит в стандартную установку Python

Формат файла tasks.json
